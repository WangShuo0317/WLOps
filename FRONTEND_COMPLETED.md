## 🚀 立即开始使用

### 方式一：一键启动（推荐）

**Windows 用户：**
```bash
cd frontend
start-frontend.bat
```

**Linux/Mac 用户：**
```bash
cd frontend
chmod +x start-frontend.sh
./start-frontend.sh
```

### 方式二：手动启动

```bash
cd frontend
npm install
npm run dev
```

### 访问地址

打开浏览器访问：**http://localhost:3000**

## 🚦 完整启动流程

### 步骤 1：启动数据库
```bash
# 确保 PostgreSQL 正在运行
```

### 步骤 2：启动后端
```bash
cd springboot-backend
mvn spring-boot:run
```

### 步骤 3：启动 Python 微服务（可选）
```bash
# 在三个不同的终端窗口中：
cd python-services/data-analyzer-service && python api.py
cd python-services/evaluation-service && python api.py
cd python-services/training-service && python api.py
```

### 步骤 4：启动前端
```bash
cd frontend
npm run dev
```

### 步骤 5：访问应用
打开浏览器访问：**http://localhost:3000**

## ✨ 使用示例

### 场景 1：标准训练流程

4. **查看结果**

## 🐛 问题排查

### 常见问题

1. **无法连接后端**
   - 确认后端是否运行在 8080 端口
   - 检查 vite.config.ts 中的代理配置

2. **依赖安装失败**
   - 尝试使用国内镜像
   - 检查 Node.js 版本

3. **页面空白**
   - 打开浏览器开发者工具
   - 查看 Console 错误信息

4. **任务状态不更新**
   - 确认后端服务正常
   - 检查 Python 微服务状态
